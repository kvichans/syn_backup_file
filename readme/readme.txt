Plugin for SynWrite.
Creates backup copy of current file: 
- by command in Plugins,
- by auto-creation on file saving.

Also gives command to call file-compare: current text, and previous backup copy.
For compare, it has option for compare-program: e.g. WinMerge (Windows).

Filename of copy - customizable via presets+variables. Variables:
- for year/month/day/hours/etc
- for parts of filename (base name, extension, parent dir, parent of parent dir...)
- for counter (with options: max value of counter, width of number)
- for upper/lower/title case

See the info, by pressing Help button in the Config dialog.
